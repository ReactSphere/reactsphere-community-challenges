# ReactSphere Weekly Challenges 🚀

Welcome to the ReactSphere Weekly Challenges!  

## How to Participate
1. Fork this repository.
2. Create a branch for your solution (e.g., `week-1-solution`).
3. Submit a Pull Request with your solution.
4. Points are tracked on the [Leaderboard](../LEADERBOARD.md).

## Points System
| Action | Points |
|--------|--------|
| PR merged | 10 |
| Challenge completed | 15 |
| Issue resolved | 5 |
| Helpful discussion comment | 2 |

New challenges will be posted here every week.
